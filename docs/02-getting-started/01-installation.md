# Installation

There are two supported ways to install Mianotes.

## Install from package

Use this if you want Mianotes installed into the normal system locations for your operating system.

- macOS: `mianotes.pkg`
- Ubuntu: `mianotes.deb`

Read: [Installing from package](01b-installation-package.md).

## Install from GitHub

Use this if you are a developer, contributor, or agent-tool builder who wants editable source checkouts of the web service and dashboard.

Read: [Installing from GitHub](01a-installation-GitHub.md).

## Which one should I choose?

Most users should start with the package installer.

Use the GitHub installer when you want to work on the code, run tests, change branches, or inspect how the services are wired together.

Read next: [First run](02-first-run.md) and [Configuration options](03-configuration.md).
